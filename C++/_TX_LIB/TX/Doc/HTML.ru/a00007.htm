Подробное описание

Класс для описания элемента диалогового окна (контрола)

Массив таких структур задает описание макета диалогового окна. Это описание похоже на задание диалога в ресурсном скрипте (.rc):

  • Нулевой элемент описывает диалоговое окно в целом
  • Остальные элементы описывают контролы
  • Последний элемент - txDialog::END или {NULL}
См. также:
txDialog::setLayout(), txDialog::dialogBox(), txDialog::dialogProc()
Примеры использования:
          Cм. реализацию функции txInputBox().
